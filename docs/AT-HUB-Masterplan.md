# Projektinformationen

Projektname: AT HUB

Projektleiter:
Alex Thiel

Projektziel:

Der AT HUB ist eine persönliche Arbeits-, Organisations- und Wissensplattform.
Er verbindet berufliche und private Organisation in einer modernen Webanwendung.

Der Arbeitsbereich und der Privatbereich bleiben vollständig getrennt.

Das Dashboard verbindet beide Bereiche ausschließlich über die Übersicht
"Was steht heute an?"

# Grundprinzipien

- Eine Datenquelle je Information

- Modulare Architektur
- Erweiterbar
- Datenschutz von Anfang an
- Mobile First
- Ein Administrator (Alex Thiel)
- Übersicht vor Komplexität
